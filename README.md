# HR Data Analysis Dashboard 📊

This Power BI dashboard analyzes HR data including:

- Employee Count
- Attrition Rate
- Department Wise Analysis
- Gender Ratio
- Salary Insights
- Performance Metrics

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- DAX

## Project File
Download `.pbix` file and open in Power BI Desktop.
